double click Build.bat 

step 2 RunServer

double click RunServer.bat

step 3 RunClient

double click RunClient.bat

step 4 enter ip and port in RunClient.bat

Enter ip: server
Enter port: 1669

step 5 enter your name

Next is how to use all command

Echo <input your text>
-Echo is command that send message back to yourself.

CreateRoom <input your roomname>
-CreateRoom is command that CreateRoom.

Say <input your text>
-Say is command that send message to everyone in your room.

Join <input your join roomname>
-Join is command that connected in room that have the same name.

Roomlist
-Roomlist is command that show all room name in this server.

Leave
-Leave is command that make leave your room.

Quit
-Quit is command that make you leave server.

Who
-Who is command that show everyone name in this server if your are in room it show only all everyone name in your room.


DM <target name> <\t> <input your text>
-DM is command that send message to target name.
